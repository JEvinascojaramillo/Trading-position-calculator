# 📊 Trading Position & Risk Calculator

An interactive web tool designed to calculate position sizes and Take Profit levels based on professional risk management rules.

### Features:
- **Dynamic Risk Management:** Calculates based on account equity and chosen risk percentage.
- **Directional Detection:** Automatically identifies if the setup is a LONG or SHORT position.
- **R/R Ratio Logic:** Suggests target profit levels based on a specific Risk/Reward ratio.
- **Responsive Design:** Optimized for mobile use, allowing you to calculate trades on the go.
